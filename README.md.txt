# Skill Swap Backend (In-Memory Demo)

This is a simple backend for Skill Swap built using Flask with in-memory data (no database).

## Features
- `GET /skills`: View all available skills
- `POST /swap_requests`: Submit a skill swap request
- `GET /swap_requests`: View all swap requests
